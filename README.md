# 🚀 全端 AI 生成個人網站模板

這是一個專為初學者設計的「全端」網頁模板。只要修改一個設定檔，整個網站的外觀和內容就能根據資料自動改變（完全不用動到前端程式碼）！

## 📚 架構簡介

本專案採用最清晰易懂的架構，分為三個主要部分：

1. **資料庫 (`config.json`)**：網站的內容來源，也是你**唯一需要修改**的檔案。
2. **後端 (`server.js`)**：使用 Node.js 建立的伺服器，負責把 `config.json` 的資料打包成 API (`/api/profile`) 傳送出去。
3. **前端 (`public/` 資料夾)**：包含網頁結構 (HTML)、樣式 (CSS) 與邏輯 (JS)，它會抓取後端資料並「動態」變成你在畫面上看到的精美排版與顏色。

---

## 🛠️ 啟動指南

1. 打開終端機 (Terminal)，確認已經在這個資料夾內。
2. 安裝後端必要的套件，輸入指令：
   ```bash
   npm install
   ```
3. 啟動後端伺服器，輸入指令：
   ```bash
   npm start
   ```
4. 打開瀏覽器，輸入 `http://localhost:3000` 欣賞你的專案！
